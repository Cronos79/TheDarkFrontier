Purpose

The Modular Asset Catalog is the master inventory of every reusable art asset created for The Dark Frontier.

Before creating a new reusable asset, this catalog should always be checked to determine whether an existing asset can be reused.

The goal is to minimize duplicate work, maintain visual consistency, and expand a cohesive modular asset library over time.

Workflow Rule

Before creating any new reusable object:

Check this catalog.
If a suitable asset already exists, reuse it.
If not, create the new asset.
Add it to this catalog immediately.

This helps keep the asset library organized, reduces duplicate work, and ensures a consistent visual style across The Dark Frontier.
